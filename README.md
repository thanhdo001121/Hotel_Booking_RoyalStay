## üî• Clone the repository

Open your terminal and then type

### `$ git clone https://github.com/dangkhoa99/CNPM.git`

## ‚ö° Installation

`cd` into the new folder and type

### `$ npm install`

## üöÄ How to run

To run the React project

### `$ npm start`

Then open another terminal and type

### `$ cd backend`

### `$ nodemon server`
